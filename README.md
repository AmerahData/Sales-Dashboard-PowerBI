Sales Performance Dashboard

Overview
An interactive sales dashboard built with Power BI 
analyzing retail sales data from the Superstore dataset.

Pages
- Overview – Sales trend & key KPIs
- Product Performance – Top products & categories
- Regional Analysis – Sales by region & state
- Customer Insights – Segments & discount impact

Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Superstore Sales Dataset

Key Insights
- Total Sales: $2.30M
- Total Orders: 5K
- Profit Margin: 12%
- Best performing region: West
